# Doodax - Advanced Base64 Utility

**The Professional's Choice for Client-Side Encoding & Decoding**

Doodax is a high-performance, privacy-focused web application built for developers, data analysts, and IT professionals. It features a stunning, immersive "Cosmic Galaxy" interface and provides instant execution of Base64 algorithms directly in the browser using the Web Crypto API.
